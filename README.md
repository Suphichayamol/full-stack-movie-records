# 🎬 Full Stack Movie Records

A Full Stack Movie Records application built with **React + Node.js + Express**.  
This project allows users to manage movie records with role-based permissions.

---

## 🚀 Features

- View movie records
- Add new movies
- Edit movie information
- Delete movies (MANAGER role only)
- Form validation
- Modal editing
- Confirm delete modal
- Role-based access control

---

## 🛠 Tech Stack

Frontend
- React
- TypeScript
- MobX

Backend

- Express
